# Sri Manju Nadh - Personal Portfolio

**🌐 Live Demo:** [View my Portfolio Here](https://personal-portfolio-nine-theta-50.vercel.app/)

Welcome to the repository of my personal portfolio website! This project serves as a dynamic, professional showcase of my skills, experiences, and projects as an AI Engineer and Software Developer.

## 🚀 Features

- **Modern & Premium UI:** Designed using modern aesthetics, featuring smooth glassmorphic effects, vibrant animated "aura" backgrounds, and elegant typography.
- **Responsive Layout:** Fully optimized for desktops, tablets, and mobile devices with a custom hamburger navigation menu.
- **Interactive Project Showcase:** Custom-built modals that dynamically load detailed information for each project, ensuring a clean and interactive browsing experience.
- **Dynamic Typing Animation:** A fluid hero section that highlights my various roles.
- **Smooth Scroll & Reveal Effects:** Content beautifully fades and slides in as you scroll down the page.
- **Continuous Skill Marquee:** An infinite scrolling marquee showcasing my technical stack.

## 🛠️ Technologies Used

The portfolio is built purely with foundational web technologies, ensuring lightweight performance and high customizability:
- **HTML5:** Semantic structure and content.
- **CSS3:** Custom styles, animations (`@keyframes`), CSS Grid & Flexbox layouts, and root variables for easy theming.
- **Vanilla JavaScript:** DOM manipulation for modals, scroll reveals, the dynamic typing effect, and navigation toggling.
